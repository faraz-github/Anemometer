# Anemometer

A digital anemometer like app made using Nodejs environment using openweathermapAPI.

## Available Commands

- npm install
- node app.js

## Installation

Anemometer requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and start the server.

```sh
cd Anemometer
npm i
node app.js
```
